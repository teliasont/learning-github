# Step 1: Create Your Account and Tutorial Copy

## Goal
Create a GitHub account and your own working copy of this tutorial.

## Instructions
1. Create a GitHub account if you do not already have one.
2. Open the original `github-starter-tutorial` repository.
3. In the repository header row (same row as **Star**), click **Use this template**.
4. Set the new repository name to **`github-startup`**.
5. Create the repository under your own account.
6. Open your new `github-startup` repository.

Do **not** click the green **Code** button for this step.
If **Use this template** is missing, make sure you are signed in and using desktop layout.

## Checklist
- [ ] I have a GitHub account.
- [ ] I created my own `github-startup` repo from the template.
- [ ] I can confirm `github-startup` is under my account.
