# Step 3: Link VS Code to GitHub

## Goal
Connect VS Code to your GitHub account so cloning, commit, and push work inside the editor.

## Instructions
1. Open VS Code.
2. Open the Extensions view (square icon in the left sidebar).
3. Search for **GitHub Pull Requests and Issues**.
4. Click **Install**.
5. In the top-right of VS Code, click the **Accounts** icon.
6. Click **Sign in with GitHub**.
7. In the browser window, click **Authorize Visual Studio Code**.
8. Return to VS Code and confirm your GitHub account appears in the Accounts menu.

## Checklist
- [ ] I installed the GitHub Pull Requests and Issues extension.
- [ ] I signed VS Code into GitHub.
- [ ] I can see my GitHub account in VS Code.
