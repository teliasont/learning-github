# Step 4: Clone `github-startup` in VS Code Source Control

## Goal
Clone your `github-startup` repository into `Documents/GitHub` using VS Code only.

## Instructions
1. Open VS Code.
2. In the left sidebar, click **Source Control**.
3. The Source Control icon looks like a split branch with connected dots.
4. Click **Clone Repository**.
5. Click **Clone from GitHub**.
6. If prompted, click **Allow**.
7. In the list under **Remote Sources**, select the `github-startup` repo you just created.
8. Choose your local `Documents/GitHub` folder as the destination.
9. If asked "Open the cloned repository?" click **Open**.
10. If VS Code asks for access permissions, click **Allow**.
11. If prompted "Do you trust the authors of the files in this folder?" choose **Yes, I trust the authors** for the parent `GitHub` folder.

Note: VS Code may suggest recommended extensions. You can click **Install** now. If you skipped the GitHub extension earlier, complete Step 3 before continuing.

## Checklist
- [ ] I opened Source Control in VS Code.
- [ ] I cloned `github-startup` from GitHub.
- [ ] I saved it in `Documents/GitHub`.
- [ ] I opened and trusted the cloned folder.
